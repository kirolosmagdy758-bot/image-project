# Image Processing Pipeline for Noisy Image Enhancement and Edge Detection

## Project Description

This project applies a complete image processing pipeline to noisy images using Python and OpenCV. The main goal is to improve image quality, reduce noise, enhance contrast, sharpen details, detect edges, and display the final processed results in an organized way.

The project uses a noisy image dataset downloaded from Kaggle using KaggleHub. The images are processed step by step using traditional image processing techniques such as grayscale conversion, filtering, CLAHE enhancement, sharpening, Sobel edge detection, Canny edge detection, segmentation, and edge overlay.

## Dataset

The dataset used in this project is:https://www.kaggle.com/datasets/goutham1208/multi-noises-for-image-denoising
